Быстрый старт

  1. Установите Node.js 18 или новее.
  2. Клонируйте репозиторий: 
    git clone https://github.com/example/Site-Prototype-Recreation.git
  3. Перейдите в каталог проекта и выполните npm install.
  4. Запустите сервер разработки: npm run dev.
  5. Для сборки продакшен версии выполните npm run build.
  6. Также вы можете воспользоваться Dockerfile для автоматической установки зависимостей.
